OpenGL = Open Graphics Library
-   created in 1992
-   will be used for this class

DirectX
-   created by Microsoft in 1995


Purpose of these Libraries:
-   removes the need to know the graphics/sound cards
-   we talk to these Libraries and the Libraries talk to the hardware for us
