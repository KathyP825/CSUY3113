In previous examples:
-   used shader program that does the drawing and filling of the polygons
    -   meant for drawing untextured polygons
        -   texture polygons have a texture on them (ex. wall, person)
-   can tell shader program to set a solid color to fill the triangle with

---------

ShaderProgram::SetColor
-   sets the color to use when drawing a polygon

Code:
ShaderProgram::SetColor(float red, float green, float blue, float alpha);      // structure

program.setColor(1.0f, 0.0f, 0.0f, 1.0f);   // set to red
