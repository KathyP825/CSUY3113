        -   stop updating / drawing / colliding / collecting that coin
    -   instead of removing coin from array and deallocating memory, set isActive bool to false

--------

Code (Entity.h):

class Entity {
public:
    EntityType entityType;
    bool isActive;      // add this bool

    glm::vec3 position;
    glm::vec3 velocity;
    glm::vec3 acceleration;
};

--------

What happens when isActive == false?

-   Update = exit right away
-   CheckCollision = always false if either object is false!
    -   Code:
            if (isActive == false || other.isActive == false) return false;
-   Render = no rendering, exit right away
